EBEAE Algorithm

Extended Blind Endmember and Abundance Extraction algorithm meant to be used to generate .png files containing a comparison between synthetic datasets and estimated results.
"A linear mixture model can represent constitutive elements (endmembers) and their contributions (abundances) per pixel of an image, EBEAE is based on a constrained quadratic optimization and an alternated least-squares strategy to jointly estimate endmembers and their abundances."
- Daniel Ulises Campos Delgado, et. al. [1]

Bill Generator

This project use .pdf and .txt files in order to generate needed and usefull information to make bills in SAT (Servicio de Administración Tributaria) platform from México Goverment. Applying Regular Expressions, tkinter and PyPDF2 modules, this is and interactive and timesaving tool for personal purposes.

Distance Blueprint

Simple implementation of Flask's blueprint using Yandex Map API in order to calculate distance from an specified address given as http request to the Moscow Ring Road (MKAD).
Note: When address is inside MKAD, distance isn't calculated.
